# ONE TAP — Market Data Dashboard

HEINEKEN Korea 내부용 오프 프리미스 시장 분석 대시보드입니다.

---

## 🚀 실행 방법

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 확인할 수 있습니다.
